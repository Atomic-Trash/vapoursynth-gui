# Keyboard Shortcuts

This document lists all keyboard shortcuts available in VapourSynth Studio.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New Project |
| `Ctrl+O` | Open Project |
| `Ctrl+S` | Save Project |
| `Ctrl+Shift+S` | Save Project As |
| `Alt+F4` | Exit Application |

## Edit Page

### Playback Controls

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Pause |
| `Home` | Go to Start |
| `End` | Go to End |
| `Left Arrow` | Step Backward (1 frame) |
| `Right Arrow` | Step Forward (1 frame) |

### Clip Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+X` | Cut Clip |
| `Ctrl+C` | Copy Clip |
| `Ctrl+V` | Paste Clip |
| `Delete` | Delete Clip |
| `S` | Split Clip at Playhead |

### In/Out Points

| Shortcut | Action |
|----------|--------|
| `I` | Set In Point |
| `O` | Set Out Point |

### Keyframes

| Shortcut | Action |
|----------|--------|
| `K` | Add Keyframe |
| `Shift+K` | Remove Keyframe |
| `[` | Previous Keyframe |
| `]` | Next Keyframe |

### Other

| Shortcut | Action |
|----------|--------|
| `T` | Add Text Overlay |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## Color Page

| Shortcut | Action |
|----------|--------|
| `C` | Toggle Compare Mode |
| `R` | Reset All Adjustments |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## Node Editor (Restore Page)

### Graph Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Save Graph |
| `Ctrl+O` | Load Graph |
| `Ctrl+G` | Generate VapourSynth Script |

### Node Operations

| Shortcut | Action |
|----------|--------|
| `Delete` | Delete Selected Node |
| `Ctrl+D` | Duplicate Selected Node |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## Tips

- Most shortcuts are context-sensitive and work when the relevant page or control is focused
- Hover over toolbar buttons to see their keyboard shortcuts in tooltips
- Use `Tab` to navigate between UI elements
- Use `Enter` to activate focused buttons

## Customization

Keyboard shortcuts are currently not customizable. This feature may be added in a future release.
